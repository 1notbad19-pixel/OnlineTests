# 🎓 OnlineTests - REST API для онлайн тестирования

## 📌 О проекте

Проект представляет собой backend-часть системы онлайн тестирования.
Разработан в учебных целях для демонстрации многослойной архитектуры
и принципов REST API.

## ✨ Возможности

- **Управление тестами**: создание, чтение, обновление, удаление
- **Фильтрация**: по категории, статусу публикации, тегам
- **Валидация**: проверка входных данных
- **In-memory хранилище**: не требует установки БД
