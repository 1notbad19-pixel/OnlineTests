#  Онлайн тесты для прохождения

##  О проекте
Веб-приложение для прохождения онлайн тестов. Пользователи могут выбирать тесты по разным темам, проходить их и сразу видеть результаты.

##  Функциональность

### Для пользователей:
-  Регистрация и вход в систему
-  Просмотр доступных тестов по категориям
-  Прохождение тестов с таймером
-  Мгновенный результат после завершения

### Для администратора:
-  Создание и редактирование тестов
-  Добавление вопросов и вариантов ответов
-  Управление пользователями
-  Просмотр общей статистики
